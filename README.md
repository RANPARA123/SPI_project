# SPI_project
This project implements a synthesizable APB-based SPI (Serial Peripheral Interface) Controller in Verilog, enabling efficient communication between an APB master and external peripheral devices. The design supports full-duplex synchronous data transfer over MOSI and MISO, with configurable clock polarity (CPOL) and phase (CPHA), allowing operation across all standard SPI modes.
The system is built as a fully integrated RTL design with a top module, combining key components such as an APB slave interface, baud rate generator, slave select control, and shift register. The APB interface uses an FSM (IDLE, SETUP, ENABLE) to manage register read/write operations, while the baud rate generator produces the SPI clock based on programmable divisors. The shift register handles serial-to-parallel and parallel-to-serial data conversion, supporting both MSB-first and LSB-first transmission.
Overall, this project demonstrates a modular and scalable SPI controller design with APB compatibility, suitable for SoC integration and digital hardware applications.
